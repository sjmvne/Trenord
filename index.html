Documentazione Tecnica

Sistema di Generazione QR

Questa applicazione genera biglietti STIBM (Sistema Tariffario Integrato Bacino di Milano) utilizzando un algoritmo di codifica basato su zone tariffarie, timestamp e checksum crittografico.

Logica di Calcolo

Prezzo: €0.80 + (numZone × €0.50)
Durata: 45 + (numZone × 15) minuti
Zone: Mi1, Mi3, Mi4, Mi5, Mi6, Mi7, Mi8, Mi9

Struttura QR Code

Il QR code è composto da tre componenti separati dal carattere :

Formato QR
BASE64_JSON:CHECKSUM:UUID

Payload JSON

app.js - generateTicket()
{
  "type": "STIBM",
  "zones": ["Mi1", "Mi3", "Mi4"],
  "from": "Mi1",
  "to": "Mi4",
  "price": "2.80",
  "duration": 105,
  "timestamp": 1709553600000,
  "valid_until": 1709559000000,
  "pnr": "ABC123DEF456"
}

Il JSON viene codificato in Base64 per garantire compatibilità con i lettori QR standard.

Checksum di Sicurezza

Generazione SHA-256
const payload = JSON.stringify(ticketData);
const base64 = btoa(payload);
const checksum = await sha256(base64 + SECRET_SALT);
const uuid = generateUUID();

// Output finale
const qrData = `${base64}:${checksum}:${uuid}`;

Il checksum SHA-256 impedisce la manipolazione dei dati del biglietto. Ogni biglietto ha un UUID univoco per tracciabilità.

Ricerca Comuni

L'app include un database di 190 comuni mappati sulle zone tariffarie STIBM, estratti dal documento ufficiale. La funzione "Da - A" permette di cercare per nome comune e calcolare automaticamente le zone necessarie.

Mapping Comuni → Zone
COMUNI_STIBM = [
  { nome: "MILANO",    zona: "Mi1" },
  { nome: "MONZA",     zona: "Mi4" },
  { nome: "LEGNANO",   zona: "Mi7" },
  ...190 comuni totali
];

// Calcolo automatico zone
fromZona = comunePartenza.zona; // "Mi1"
toZona   = comuneArrivo.zona;   // "Mi7"
// → Slider aggiornato: Mi1 | Mi7

Informazioni

BIGLIETTO INTEGRATO STIBM
Vale per corse illimitate, sui treni e su tutti i mezzi di trasporto pubblico del bacino Milano e Monza Brianza, nel corso del periodo di validità che dipende dal numero di zone acquistate. Non ha una data di utilizzo prestabilita, ma deve essere attivato in modalità Tap&Go all'inizio del primo viaggio.

Attivazione dei titoli di viaggio

Biglietti
Se acquistati per una data specifica si attivano automaticamente all'orario di partenza prescelto.
Se a data aperta devono essere attivati tramite la funzionalità Tap&Go.

Abbonamenti
Se caricati su tessera lo Viaggio devono essere attivati presso le convalidatrici dal giorno successivo a quello di acquisto ed entro il numero di giorni indicato sulla mail ricevuta dopo l'acquisto.
Su PhonePass non è necessaria attivazione.

Carnet
Prima di salire a bordo è necessario attivare ogni viaggio, scegliendo la direzione della corsa, tramite la funzionalità Tap&Go.
